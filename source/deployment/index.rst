==============================
Deploying Plone for Production
==============================

.. admonition:: Description

    A guide to the Plone deployment stack, including load balancing, proxy caching,
    server preparation, backups, log rotation, and process control.

    This guide particularly focuses on Unix-workalike environments, though
    the stack discussion may be useful to everyone.
